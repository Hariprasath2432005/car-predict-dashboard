# 🚗 Car Price Prediction Dashboard — ML + React

A full-stack car price prediction application with an interactive 
React dashboard and Machine Learning model integration.

## ✨ Features
- 🚗 Car price prediction using ML model
- ⚛️ React + Vite modern frontend
- 📊 Interactive prediction dashboard
- 🎨 Beautiful UI with Tailwind CSS & shadcn-ui
- 📈 Data preprocessing & model training
- 🔍 EDA & data visualization
